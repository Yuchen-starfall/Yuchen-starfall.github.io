Foundation of R

注意

  • 快捷键：⌥ +- 为->

  • 在调用了dplyr之后，使用管道符大快捷键： Cmd + Shift + M

  • 大小写敏感

  • 命名不能有空格

  • 英文逗号隔开

  • 等差数列s1 <- seq(from = 1,to = 100,by = 1)

  • 重复(repeat)s2 <- rep(x = c(0, 1), times = 3)

  • 逻辑型常称为布尔型（Boolean）， 它的常量值只有 TRUE 和 FALSE

  • 因子型，带层级

数据结构

矩阵

  • 默认优先竖排；想要横排加入byrow = TRUE

  • Array（矩阵）三大属性的查询

    class() 属性

    length() 长度

    dim() 纬度

列表

list()函数创建列表,可以集成各种不同的数据类型与结构

数据框

特殊的列表

优点

  • 可以实现列表输出数据，像excel中的表格一样

运算符

求余 %%

相除求商 %/%

向量的点乘 %*%

矩阵相乘 %*%

指数 ^

循环补齐

做算术运算时，当两个向量长度相等的时候，就一一对应的完成计算；当两个向量长度不相等的时候，短的向量会循环补齐，保持与长向量的长度一致后，再做运算

关系运算符

关系运算符——特征输出结果是布尔值，及 TRUE 或 FALSE

< > == <= >= !=(不等于)

逻辑运算符

&（对所有两组数据，两个都是TRUE就输出TRUE）

｜（对所有两组数据，两个中有一个为TRUE就输出TRUE）

！（取反）

&& （对两组数据的第一个数据，两个都是TRUE就输出TRUE）

｜｜（对两组数据的第一个数据，两个中有一个为TRUE就输出TRUE

Other operators

a <- 1:10 输出1至10间的所有整数，创建一个单精度整数序列

b <- seq(from = 1, to = 10, by = 1) 输出一个等差数列，1开始，10结束，公差为1。为双精度数值型

c(2, 3, 7) %in% c(1, 2, 3, 4, 5) 判断元素是否属于向量

判断是否为缺失值（NA)

Special values

Inf，是Infinity的简写，表示无限大；-Inf表示无限小

NaN，是Not a Number的简写，表示这个数字没有数学定义

NA，是Not available的简写，表示缺失状态。

NULL，是No value的意思，表示没有值，或者空值的意思，表示变量实际上没有任何值，或者甚至不存在

函数

  • 求开方函数sqrt(x)
  • 对数函数 log(x)
  • 求均值 mean(x)
  • 标准差 sd(x)
  • 从大到小排序 sort(x)
  • 去重复 unique(x)
  • 分位数，也就是概率统计中的分位点 quantile(x)
  • 判断 is.numeric(x) is.character(x)
  • 转化函数 as.numeric(x) as.character(x)
  • 大小计算函数x>5 等，但仅仅输出TRUE或者FALSE
    • ifelse(x > 5, "big", "small") 可以改变输出值
  • 自定义函数
  • 在代码最后一行，可以用return返回计算后的值。当然，如果函数的目的只是返回最后一行代码计算的值，这个return可以省略。
  • 函数调用，现在这个函数名字叫my_std，需要用这个函数的时候，就调用它的名字my_std()。
  • 宏包函数
    1. 先安装宏包（仅安装一次就可以后继续使用）library(宏包名字)
    2. 调用函数（注意使用::指定函数的所属宏包）例如：dplyr::select()

注意

  • 返回值return()的灵活应用

  • 使用if- else if-else的条件语句

  • 如果返回值为多个结果，把结果先放在list或者data.frame中再返回

子集选择（切片）

对象就是我们在计算机里新建了存储空间，好比一个盒子， 我们可以往盒子里面装东西（赋值），可以查看里面的内容或者对里面的内容做计算（函数），也可以从盒子里取出部分东西（子集选取）

选择指定向量的位置——正整数

删除指定位置——负整数

提取TURE对应值——逻辑向量

常用与筛选大于某个值的向量

返回对应位置的向量——命名向量（like 键值对）

我们可以用命名向量，返回其对应位置的向量

提取列表值——$

提取矩阵——[]

我们取第1行到第2行的2-3列，[1:2, 2:3]，中间以逗号分隔，于是得到一个新的矩阵

默认情况下, [ 会将获取的数据，以尽可能低的维度形式呈现。

  • Like a matrix
