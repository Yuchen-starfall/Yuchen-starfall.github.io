How to quickly install and manage packages on mac ？

许多时候我们需要从第三方下载应用，他们似乎特别喜欢自作主张，逮到一个位置就落地生根，日后你想要升级、卸载都不知道去哪里找它们。此时可以使用HomeBrew，它作为一个包管理工具，可以有效的管理软件啊安装、升级等功能

但在Mac OS X 10.11系统以后，/usr/local/等系统目录下的文件读写是需要系统root权限的，而Homebrew安装如果没有指定安装路径，会默认安装在这些需要系统root用户读写权限的目录下，导致有些命令需要添加sudo前缀来执行。sudo chown -R $USER /usr/local

Command：

概念用途安装内容安装命令安装位置
Formula源码编译开源工具/库命令行程序brew install <formula>$(brew --prefix)/Cellar
Cask安装 macOS 应用/字体GUI 应用brew install --cask <cask>/Applications 或 ~/Applications

Shell – ohmyzsh

如何与计算机交互？Shell应运而生，Shell其实就是字面意思–壳. 一个系统有内核(Kernel)（“内核”指的是一个提供硬件抽象层、磁盘及文件系统控制、多任务等功能的系统软件。 内核是操作系统最基本的部分）自然就有外壳(Shell). 在 Linux 上广义的 Shell 就是与用户交互的界面, 图形化界面是 Shell, 命令行界面也是 Shell. 狭义上的 Shell 就是 Linux 上的命令行界面. 一般发行版上的命令行界面是 bash。Shell 是用来执行计算机程序的. 例如执行 mkdir 就是创建一个文件夹, 执行 touch 就是创建一个文件. 执行 ps 就是查看进程信息. 其中 mkdir, touch, ps 都是用 C 语言编写编译后的程序. 如今最常用的shell是bash和zsh

notice

其实在之前的Linux的blog里面所提到的命令都是狭义上的shell

1989年 Bash 横跨出世，作为Linux默认Shell，macOS 2019年前亦默认采用。1990年至今，作为「终极Shell」，Zsh被开发出来，同时2019年起成为macOS默认Shell。但是由于配置过于复杂，所以初期无人问津，很多人跑过来看看 zsh 的配置指南，什么都不说转身就走了。直到有一天，国外某位大牛开源出一个能够让你快速上手的 zsh 项目，叫做『oh my zsh』，使得 zsh 逐渐流行起来。

在 ~/.zshrc 设置 ZSH_THEME="powerlevel10k/powerlevel10k"。接下来，终端会自动引导你配置 powerlevel10k。

插件：

  • zsh -autosuggestions

    • 命令提示插件，当你输入命令时，会自动推测你可能需要输入的命令，按下右键可以快速采用建议

    • 插件下载到本地的 ~/.oh-my-zsh/custom/plugins

    • git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

# 无法使用github可以使用下面任意一个加速下载
# 加速1
git clone https://github.moeyy.xyz/https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
# 加速2
git clone https://gh.xmly.dev/https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
# 加速3
git clone https://gh.api.99988866.xyz/https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

  • z

    • 内置了 z 插件。z 是一个文件夹快捷跳转插件，对于曾经跳转过的目录，只需要输入最终目标文件夹名称，就可以快速跳转，避免再输入长串路径

  • extract

    • 用于解压任何压缩文件，不必根据压缩文件的后缀名来记忆压缩软件。使用 x 命令即可解压文件

最终激活：修改~/.zshrc中插件列表：plugins=(git zsh-autosuggestions zsh-syntax-highlighting z extract)

Interesting software

Thefuck

终端命令自动修复工具.当你在终端中输入的命令出现错误时，你不用细致思考该如何修正，大喊一声【fuck】（其实是在终端输入），命令就自动改正并正确执行.

Tldr-pages

长时间不使用某些命令难免会有生疏，此时如果使用man命令可能会让你崩溃，因为文档实在是Too Looooong。那么有没有解决方案呢？其中比较有名的就是tldr

tldr=Too Long; Didn’t Read，它简化了烦琐的man指令帮助文档，仅列出常用的该指令的使用方法。相比较man给出完整的帮助文档而言，大多数情况下，给出几个指令的使用demo可能正是我们想要的。
