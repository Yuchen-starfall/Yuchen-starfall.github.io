Polymorphism is often defined as the presence of more than one genetically distinct type in a single population.

Intuitive statistics

  • Number of alleles：More alleles, larger diversity;

  • Minor allele frequency (MAF)：is the frequency of the less (or least) frequent allele in a given locus and a given population.

Mutation: MAF ≤1%

Polymorphism: MAF >1%

Heterozygosity

用于衡量群体中某个位点的遗传多样性，通常分为 预期杂合度（Expected Heterozygosity, He） 和 观测杂合度（Observed Heterozygosity, Ho）

Neutral hypothesis （中性学说）

数学本质：由于抽样误差导致的概率变化

  • 演化的 drive-force 是选择、迁移、random drift
  • 有害、中性的突变是多数
  • 大部分新突变消失，少量新突变被固定依赖random drift

Tajima’s Test 用于检测DNA序列数据是否符合中性进化理论（即遗传变异仅由突变和随机遗传漂变驱动，无自然选择或群体结构变化的影响）。其核心是通过比较两种不同的群体突变率（θ）估计值——基于平均核苷酸差异（π）和基于分离位点数（S）的Watterson估计值（θ_W），从而推断是否存在自然选择或群体历史事件。
