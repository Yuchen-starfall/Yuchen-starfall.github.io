计算加性效应（a）

加性效应反映的是等位基因的累积效应，即：$a = \frac{{P_{AA} - P_{aa}}}2$

  • 逻辑：加性效应衡量的是AA 和 aa 之间的遗传贡献的差异，但由于生物学上基因型是两条染色体上的等位基因决定的，因此我们将其差异平分到两个等位基因上。
  • 结果解读：
    • a > 0 → 说明 A 等位基因对性状的正向贡献更大。
    • a < 0 → 说明 a 等位基因对性状的正向贡献更大。

计算显性效应（ d ）

显性效应反映的是杂合子（Aa）相对于两种纯合子（AA 和 aa）的偏离程度：$d = P_{Aa} - \frac{{P_{AA} + P_{aa}}}{2}$

  • 逻辑：杂合子（Aa）的表型如果完全由加性效应决定，它的表现应该是 AA 和 aa 的中间值,但如果 杂合子（Aa）偏离了这个中间值，就说明它受到了显性效应的影响， d 反映的就是这个偏移量。

显性程度（ d/a ）

(d/a) 值 解释
( d/a = 0 ) 完全加性（杂合子正好是两种纯合子的均值）
( 0 < d/a < 1 ) 部分显性（杂合子更接近显性纯合子，但仍介于两者之间）
( d/a = 1 ) 完全显性（杂合子和显性纯合子表现完全相同）
( d/a > 1 ) 超显性（杂合子表现超过显性纯合子）
( d/a < 0 ) 负显性（杂合子表现比两个纯合子都低）

聚类

  • A型系统：
    • 主要基于籼稻亚种内杂交（indica × indica）。
    • 代表品种包括早期的三系杂交水稻（如珍汕97A）。
  • B型系统：
    • 基于籼稻亚种内杂交，但引入了两系法（光温敏不育系，如PA64S）。
    • 代表品种包括“两优培九”（LYP9）。
  • C型系统：
    • 基于籼-粳亚种间杂交（indica × japonica）。
    • 利用广亲和基因克服亚种间杂交障碍，代表品种包括一些高产籼粳杂交稻。
特征 A型系统 B型系统 C型系统
亲本类型 籼稻 × 籼稻 籼稻 × 籼稻（两系法） 籼稻 × 粳稻
遗传基础 部分显性（如Hd3a、TAC1） 部分显性 + 母本特殊等位基因 超显性 + 伪超显性（如OsSPL14）
杂种优势 中等 较高 最高
应用范围 籼稻主产区 籼稻主产区 籼-粳混栽区
技术难点 杂种优势有限 光温敏不育系稳定性 生殖隔离障碍
