## Foundation

### Classification

按照化学组成

* 丙三醇（甘油） + 3 脂肪酸（Fatty acyls） --> 甘油三酯（储存能量、氧化供能）
  * 据不同脂肪酸分简单/混合甘油三酯（triacylglycerol）
  * 脂肪酸种类决定甘油三酯的熔点。链长、数目增加而增加
* 丙三醇 + 2 脂肪酸 + （磷酸 + 含氮化合物） --> 甘油磷脂（生物膜脂质双分子层）
* 鞘氨脂 + 脂肪酸 + 磷酸胆碱 --> 神经鞘磷脂（生物膜重要成份）
* 鞘氨脂 + 脂肪酸 + 糖  --> 鞘糖脂（参与细胞识别与信息传递）
* 胆固醇 + 脂肪酸 --> 胆固醇脂（转化为胆汁酸、类固醇激素参与物质代谢调节）

### Metabolism of triacylglycerol

1. 脂肪动员（fat mobilization）

   1. 关键酶：激素敏感性甘油三酯**脂肪酶**
   2. 脂解激素：胰高血糖素、肾上腺素、去甲肾上腺素。抗脂解激素：胰岛素、前列腺素E2
   3. 禁食、饥饿、交感神经兴奋

2. 甘油代谢

   1. 甘油 + 甘油激酶 （**消耗ATP过程**） --> 3-磷酸甘油 + 磷酸甘油脱氢酶 --> 磷酸二羟丙酮

   2. 脂肪酸β-氧化
      1. 脂肪酸 + CoA-SH + 脂酰CoA合成酶（Mg + ATP）--> （内质网及线粒体外膜）脂酰CoA
      2. 脂酰CoA进入线粒体（肉碱转移酶 I）**限速步骤**
      3. 脂酰CoA氧化分解（脱氢、加水、再脱氢、硫解）为乙酰CoA
      4. 乙酰CoA --> 三羧酸循环（彻底氧化）/酮体（肝外组装）
      
      以16碳软脂酸的氧化为例：消耗两个ATP活化，7轮β-氧化产生8分子乙酰CoA、7分子NADH+H、7分子FADH2。8×10 + 7×2.5 + 7×1.5 -2 = 108 -2 =106

3. 酮体（乙酰乙酸(acetoacetate) 、β-羟丁酸(β-hydroxybutyrate)、丙酮(acetone)）

   1. 合成： 2分子乙酰CoA + 乙酰CoA硫解酶 --> 乙酰乙酰CoA + 乙酰CoA + HMGCoA合成酶 --> HMGCoA + HMGCoA裂解酶 -->  酮体互变

   2. 利用：β-羟丁酸 + β-羟丁酸脱氢酶  -->  乙酰乙酸 + 琥珀酰CoA -->  乙酰乙酰CoA + 琥珀酸 + HSCoA  -->  乙酰CoA

4. 脂肪酸合成

   1. 线粒体内乙酰CoA柠檬酸-丙酮酸循环出线粒体
   2. ATP + HCO3- + 乙酰CoA + 乙酰CoA**羧化酶**（限速酶，辅酶生物素） --> **丙二酰CoA**（合成单体） + ADP + Pi
   3. 缩合（合成酶）、加氢（还原酶）、脱水（脱水酶）、再加氢（还原酶）

5. 胆固醇合成

   1. 甲醛戊酸合成（HMG-CoA还原酶（限速），内质网，昼夜节律--午夜最高、中午最低）、鲨烯合成、胆固醇合成

   2. 调控：SCAP-SREBP（胆固醇调节元件结合蛋白）促进合成（它可以在ER上感知胆固醇的含量，如果胆固醇含量较少，便可启动胆固醇的合成）

   3. 转变胆汁酸、类固醇激素、皮肤：7-脱氢胆固醇 + UV --> 维生素D3


## Quize

### 三类降血胆固醇药及其作用机制：

①他汀类药物：抑制HMG-CoA还原酶，降低胆固醇的合成。

②Ezetimibe：抑制NPC1L1(NPC1L1 的作用：在小肠中作为食物胆固醇吸收的受体)，抑制从食物中吸收胆固醇。

③PCSK9抑制剂：提高LDLR 稳定性，减少血液中LDL。
